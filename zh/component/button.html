Button 按钮 #

常用的操作按钮。

基础用法 #

使用 type、plain、round 和 circle 来定义按钮的样式。

查看源代码
<template>
  <div class="mb-4">
    <van-button>Default</van-button>
    <van-button type="primary">Primary</van-button>
    <van-button type="success">Success</van-button>
    <van-button type="info">Info</van-button>
    <van-button type="warning">Warning</van-button>
    <van-button type="danger">Danger</van-button>
  </div>
  <div class="mb-4">
    <van-button plain>Plain</van-button>
    <van-button type="primary" plain>Primary</van-button>
    <van-button type="success" plain>Success</van-button>
    <van-button type="info" plain>Info</van-button>
    <van-button type="warning" plain>Warning</van-button>
    <van-button type="danger" plain>Danger</van-button>
  </div>

  <div class="mb-4">
    <van-button round>Round</van-button>
    <van-button type="primary" round>Primary</van-button>
    <van-button type="success" round>Success</van-button>
    <van-button type="info" round>Info</van-button>
    <van-button type="warning" round>Warning</van-button>
    <van-button type="danger" round>Danger</van-button>
  </div>

  <div>
    <van-button icon="search" circle />
    <van-button type="primary" icon="edit" circle />
    <van-button type="success" icon="check" circle />
    <van-button type="info" icon="message" circle />
    <van-button type="warning" icon="star" circle />
    <van-button type="danger" icon="delete" circle />
  </div>
</template>
<style>
.mb-4 {
  margin-bottom: 20px;
}
</style>

禁用状态 #

你可以使用 disabled 属性来定义按钮是否被禁用。

使用 disabled 属性来控制按钮是否为禁用状态。 该属性接受一个 Boolean 类型的值。

查看源代码
<template>
  <div class="mb-4">
    <van-button disabled>Default</van-button>
    <van-button type="primary" disabled>Primary</van-button>
    <van-button type="success" disabled>Success</van-button>
    <van-button type="info" disabled>Info</van-button>
    <van-button type="warning" disabled>Warning</van-button>
    <van-button type="danger" disabled>Danger</van-button>
  </div>

  <div>
    <van-button plain disabled>Plain</van-button>
    <van-button type="primary" plain disabled>Primary</van-button>
    <van-button type="success" plain disabled>Success</van-button>
    <van-button type="info" plain disabled>Info</van-button>
    <van-button type="warning" plain disabled>Warning</van-button>
    <van-button type="danger" plain disabled>Danger</van-button>
  </div>
</template>

Basic text button

Disabled text button

<template>

  <div class="flex justify-space-between mb-4 flex-wrap gap-4">
    <van-button
      v-for="button in buttons"
      :key="button.text"
      :type="button.type"

    >

调整尺寸 #

除了默认的大小，按钮组件还提供了几种额外的尺寸可供选择，以便适配不同的场景。

使用 size 属性额外配置尺寸，可使用 large和small两种值。

查看源代码
<template>
  <div>
    <van-button size="large">Large</van-button>
    <van-button>Default</van-button>
    <van-button size="small">Small</van-button>
    <van-button size="large" round>Large</van-button>
    <van-button round>Default</van-button>
    <van-button size="small" round>Small</van-button>
  </div>
</template>

自定义颜色 #

您可以自定义按钮颜色。

我们将自动计算 hover 和 active 颜色。

查看源代码
<template>
  <div class="flex">
    <van-button color="#626aef">Default</van-button>
    <van-button color="#626aef" plain>Plain</van-button>

    <van-button color="#626aef" disabled>Disabled</van-button>
    <van-button color="#626aef" disabled plain
      >Disabled Plain</van-button
    >
  </div>
</template>
