Checkbox 多选框 #

在一组备选项中进行多选。

基础用法 #

单独使用可以表示两种状态之间的切换，写在标签中的内容为 checkbox 按钮后的介绍。

checkbox-group元素能把多个 checkbox 管理为一组，只需要在 Group 中使用 v-model 绑定 Array 类型的变量即可。 只有一个选项时的默认值类型为 Boolean，当选中时值为true。 van-checkbox 标签中的内容将成为复选框按钮之后的描述。

<template>
  <div>
    <van-checkbox v-model="checked1" label="Option 1" />
    <van-checkbox v-model="checked2" label="Option 2" />
  </div>
  <div>
    <van-checkbox v-model="checked3" label="Option 1" />
    <van-checkbox v-model="checked4" label="Option 2" />
  </div>
  <div>
    <van-checkbox v-model="checked5" label="Option 1" />
    <van-checkbox v-model="checked6" label="Option 2" />
  </div>
  <div>
    <van-checkbox v-model="checked5" label="Option 1" disabled />
    <van-checkbox v-model="checked6" label="Option 2" disabled />
  </div>
</template>

<script lang="ts" setup>
import { ref } from 'vue'

const checked1 = ref(true)
const checked2 = ref(false)
const checked3 = ref(false)
const checked4 = ref(false)
const checked5 = ref(false)
const checked6 = ref(false)
</script>

禁用状态 #

多选框不可用状态。

设置 disabled 属性即可。

<template>
  <van-checkbox v-model="checked1" disabled>Disabled</van-checkbox>
  <van-checkbox v-model="checked2">Not disabled</van-checkbox>
</template>

<script lang="ts" setup>
import { ref } from 'vue'
const checked1 = ref(false)
const checked2 = ref(true)
</script>

多选框组 #

适用于多个勾选框绑定到同一个数组的情景，通过是否勾选来表示这一组选项中选中的项。

在 van-checkbox 元素中定义 v-model 绑定变量，单一的 checkbox 中，默认绑定变量的值会是 Boolean，选中为 true。 在 van-checkbox 组件中，label 是选择框的值。 如果该组件下没有被传入内容，那么 label 将会作为 checkbox 按钮后的介绍。 label 也与数组中的元素值相对应。 如果指定的值存在于数组中，就处于选择状态，反之亦然。

<template>
  <van-checkbox-group v-model="checkList">
    <van-checkbox label="Option A" />
    <van-checkbox label="Option B" />
    <van-checkbox label="Option C" />
    <van-checkbox label="disabled" disabled />
    <van-checkbox label="selected and disabled" disabled />
  </van-checkbox-group>
</template>

<script lang="ts" setup>
import { ref } from 'vue'

const checkList = ref(['selected and disabled', 'Option A'])
</script>
