Table 表格 #

用于展示多条结构类似的数据， 可对数据进行排序、筛选、对比或其他自定义操作。

基础表格 #

基础的表格展示用法。

当 van-table 元素中注入 data 对象数组后，在 van-table-column 中用 prop 属性来对应对象中的键名即可填入数据，用 label 属性来定义表格的列名。 可以使用 width 属性来定义列宽。

<template>
  <van-table :data="tableData" style="width: 100%">
    <van-table-column prop="date" label="Date" width="180" />
    <van-table-column prop="name" label="Name" width="180" />
    <van-table-column prop="address" label="Address" />
  </van-table>
</template>

<script lang="ts" setup>
const tableData = [
  {
    date: '2016-05-03',
    name: 'Tom',
    address: 'No. 189, Grove St, Los Angeles',
  },
  {
    date: '2016-05-02',
    name: 'Tom',
    address: 'No. 189, Grove St, Los Angeles',
  },
  {
    date: '2016-05-04',
    name: 'Tom',
    address: 'No. 189, Grove St, Los Angeles',
  },
  {
    date: '2016-05-01',
    name: 'Tom',
    address: 'No. 189, Grove St, Los Angeles',
  },
]
</script>

带斑马纹表格 #

使用带斑马纹的表格，可以更容易区分出不同行的数据。

固定表头 #

只要在 van-table 元素中定义了 height 属性，即可实现固定表头的表格，而不需要额外的代码。

自定义列模板 #

自定义列的显示内容，可组合其他组件使用。

通过 slot 可以获取到 row, column

<template>
  <van-table :data="tableData" style="width: 100%">
    <van-table-column label="Date" width="290">
      <template #default="scope">
        <div style="display: flex; align-items: center">
          <VanIcon name="timer" />
          <span style="margin-left: 10px">{{ scope.row.date }}</span>
        </div>
      </template>
    </van-table-column>
    <van-table-column label="Name" width="180">
      <template #default="scope">
        <VanTooltip effect="light" trigger="hover" placement="top" width="auto">
          <VanButton>{{ scope.row.name }}</VanButton>
          <template #content>
            <div>name: {{ scope.row.name }}</div>
            <div>address: {{ scope.row.address }}</div>
          </template>
        </VanTooltip>
      </template>
    </van-table-column>
    <van-table-column label="Operations">
      <template #default="scope">
        <van-button size="small" @click="handleEdit(scope.$index, scope.row)"
          >Edit</van-button
        >
        <van-button
          size="small"
          type="danger"
          @click="handleDelete(scope.$index, scope.row)"
          >Delete</van-button
        >
      </template>
    </van-table-column>
  </van-table>
</template>

<script lang="ts" setup>
import { VanIcon, VanTooltip, VanButton } from '@vangle/components'

interface User {
  date: string
  name: string
  address: string
}

const handleEdit = (index: number, row: User) => {
  console.log(index, row)
}
const handleDelete = (index: number, row: User) => {
  console.log(index, row)
}
const print = (row: any) => {
  console.log(row)
  return row
}
const tableData: User[] = [
  {
    date: '2016-05-03',
    name: 'Tom',
    address: 'No. 189, Grove St, Los Angeles',
  },
  {
    date: '2016-05-02',
    name: 'Tom',
    address: 'No. 189, Grove St, Los Angeles',
  },
  {
    date: '2016-05-04',
    name: 'Tom',
    address: 'No. 189, Grove St, Los Angeles',
  },
  {
    date: '2016-05-01',
    name: 'Tom',
    address: 'No. 189, Grove St, Los Angeles',
  },
]
</script>
